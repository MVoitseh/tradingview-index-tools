# TradingView Index Tools (Pine Script)

Portfolio of professional-grade, session-based tools for index traders.
Focus: Initial Balance (IB), extensions, stable alerts, clean object management, and non-repainting logic.

## Products
- **DAX Initial Balance Pro (London IB 09:00–10:00)**  
  IBH/IBL/Mid, ±1x/±2x extensions, clean multi-day history, Close/Wick alerts, anti-spam.

- **NASDAQ Initial Balance Pro (US Cash IB 09:30–10:30)**  
  IB engine + extensions, session-safe architecture, reliable breakout alerts, once-per-day guard.

- **IB Extension Addon Pro**  
  Standalone IB alert engine: IB breaks, ±1x (optional ±2x), reclaim detection, Close/Wick modes.

## What these scripts are built for
- Accurate session calculations (exchange time)
- Non-repainting logic / stable historical levels
- Clean chart UX (controlled objects, no clutter)
- Reliable alerts (Close/Wick modes, anti-spam, once-per-day guards)
