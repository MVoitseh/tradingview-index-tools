# DAX Initial Balance Pro (London IB)

**Session:** 09:00–10:00 (Exchange Time)  
**Market:** DAX / GER40 / FDAX

## Features
- IB High / IB Low / 50% Mid
- Range-based extensions: ±1x and ±2x
- Strict IB box during IB window
- Multi-day history control (keep last N days)
- Stable bar_index anchoring (no drifting)
- Alerts: IB formed, IBH/IBL breaks, extension breaks
- Optional reclaim + Close/Wick modes + anti-spam / once-per-day guards
- Non-repainting, performance-safe, no warnings
