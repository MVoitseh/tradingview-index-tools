# IB Extension Addon Pro

Standalone IB alert engine built for session-based breakout workflows.

## Features
- Automatic IB calculation (session-based)
- Alerts: IBH/IBL breaks, ±1x extensions (optional ±2x)
- Reclaim detection + Close/Wick trigger modes
- Anti-spam controls + daily-stable behavior
- Non-repainting architecture, performance-safe
